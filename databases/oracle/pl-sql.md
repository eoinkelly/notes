## Pl/SQL

* Procedural extensions to SQL
* Basically a full programming language

https://en.wikipedia.org/wiki/PL/SQL


* DML = data manipulation language
    * i.e. SQL
    * allows CRUD of data
        * C = INSERT ...
        * R = SELECT ...
        * U = UPDATE ...
        * D = DELETE ...
* DDL = data definition language
    * In SQL:
        * CREATE
        * DROP
        * ALTER
        * RENAME
        * TRUNCATE
    * In XML:
        * XML Schema
