# System for Cross Domain Identity Management


* Pronounced "skim"
* A Protocol
* Rest + JSON
* Common set of Rest API requests for CRUDing and searching identities
* Kind of a "JSON:API for identity management" idea
* History
* Versions
    * SCIM 1.1
        * Created 2011
    * SCIM 2.0
        * Created 2015
